# 📸 Image Gallery App

A modern and responsive Image Gallery Web Application built using **HTML, CSS, and JavaScript**.
This project showcases a clean UI with interactive features like filtering and lightbox view.

---

## 🚀 Features

* 🖼️ Responsive image grid layout
* 🔍 Filter images by category (Nature / Tech / All)
* 🔎 Lightbox view (click image to enlarge)
* 🎨 Hover effects with smooth animations
* 📱 Fully responsive for mobile & desktop

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (DOM Manipulation)
